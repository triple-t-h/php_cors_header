# AMP CORS header with PHP

Set CORS header in AMP with PHP. [AMP developer DOCUMENTATION](https://amp.dev/documentation/guides-and-tutorials/learn/amp-caches-and-cors/amp-cors-requests?referrer=ampproject.org "AMP Caches & CORS").
## Simple Example
```php
<?php
include_once('your/path/AMPRequest.php');

AMPRequest::setCORSHeader('antik--lübz-de');
?>
```
